Workflows

Node.js

You can use the process function with 2 parameters:

  • css is the CSS as string
  • options is an object specifying options you want

It returns fixed CSS as string.

var pleeease = require('pleeease'),
    fs       = require('fs');

var css = fs.readFileSync('app.css', 'utf8');

// define options here
var options = {};

var fixed = pleeease.process(css, options);

fs.writeFile('app.min.css', fixed, function (err) {
  if (err) {
    throw err;
  }
  console.log('File saved!');
});

With Brunch

If you're using Brunch (good point), you may want to use pleeease-brunch.

Add this plugin as a dependency in your package.json

{
  "dependencies": {
    "pleeease-brunch": "0.2.x"
  }
}

Or install it:

$ npm install --save pleeease-brunch

Now, Pleeease-brunch does its job. You can set it in Brunch configuration file:

plugins:
  pleeease:
      autoprefixer: true
      minifier: true
      mqpacker: true
      fallbacks:
          variables: true
          rem: false
          pseudoElements: false

Note that pleeease-brunch is an optimizer plugin, so it only works when optimize: true is set. The --production environment activates optimizers.

You can also use the pre-configured Pleeease skeleton.
